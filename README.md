## Installation

```console
# clone the repo
$ git clone https://github.com/JPAntonisse/nox-spellenzolder-rating.git

# install the requirements
$ python3 -m pip install -r requirements.txt
```

## Usage

```console
$ python3 noxrating --api_key {YOUTUBE_API_KEY}
```
